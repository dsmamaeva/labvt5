Лабораторные печи
Линейка настольных печей с объемом рабочего пространства от 3 до 100 литров. Широкий набор опций позволяет использовать печь для процессов с удалением летучих веществ, купелирования, закалки, отжига и нормализации.
Для процессов с температурами до 1300 °С применяется нагрев с помощью проволочных нагревателей, для эксплуатации с температурами до 1400 °С нагреватели из карбида кремния.
Основные возможности:Корпус и дверь выполнены с воздушным зазором для уменьшения температуры наружных поверхностей печи
Распашная дверь с механизмом отключения нагрузки при открытии
Вентиляционное отверстие в своде
ПИД управление нагревом
Соответствие стандартам электробезопасности
Максимальная температура, °С
1200, 1300, 1400
Конструктивное исполнение
нагревательных элементов
Два варианта исполнения нагревательных элементов:
• Размещение нагревательных элементов в штробе
• Закрытое исполнение (нагревательные элементы закрыты от рабочего объема)
Регулятор температуры
Два варианта оснащения регуляторами с возможностями:
• ПИД регулирование по уставке
• ПИД регулирование по программе (внесен в реестр средств измерений Российской Федерации)
